# Admin.NET.Ai 核心功能手册

Admin.NET.Ai 是一个强大的本地化 AI Agent 搭建底座，具有以下核心特性：

1. **多模型工厂 (AiFactory)**：提供统一的接口管理多个大语言模型提供商（如 OpenAI, 智谱清言, 阿里百炼, DeepSeek 等），支持动态切换和自动回退机制。
2. **中间件管道**：采用类似 ASP.NET Core 的洋葱模型，内置了请求日志、内容审计（防护栏）、重试机制、限流策略和 Token 消耗统计监控。
3. **基础 RAG 知识检索**：内建了朴素 RAG（利用向量计算相似度）和基于知识图谱 GraphRAG（多跳推理）的检索能力。
4. **MCP 协议支持**：完美融合 Model Context Protocol (MCP)，支持通过 Stdio 或 HTTP SSE 的方式，让大模型挂载本地服务器的外部工具和资源。
5. **结构化输出 (Structured Output)**：通过 JSON Schema 严格约束大模型的输出格式，使其直接返回后端业务可用的实体类对象，减少解析异常。
6. **多智能体协作**：支持编排复杂的 Agent 工作流，包含并行执行、评审循环和状态共享。
